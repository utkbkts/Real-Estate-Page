Merhaba bu proje'de routeları kullanarak sayfaları yönlendirdim navlink kullanarak active classı ekledim sayfalar arası geçişleri kullanıcının sayfayı daha kullanışlı bir şekilde kullanmasını sağladım.NotFound sayfası yaptım eğer kullanıcı site üzerinden farklı bir id'ye gitmeye çalışırsa bu sayfayı görecektir.
Bradcrump geçişleri için uselocation kullandım pathname ile ulaştım split ile aralarına slash koydum filterledım mapledim += ile fazla olursa linklerleri 
home > about şeklinde göstermesini sağladım.


Hello, in this project, I redirected the pages using routers, I added the active class using navlink, I made the transitions between pages so that the user can use the page in a more useful way. I made a NotFound page, if the user tries to go to a different id on the site, he will see this page.
I used uselocation for bradcrump transitions, I reached with pathname, I put slash between them with split, I filtered, I mapped, if there are more with +=, I mapped the links. 
I made it show as home > about.
